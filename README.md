# Webスクレイピングサンプル1

## 概要
webdriver の自動インストール  
Chromeを操作してログイン  
テーブルデータの抽出  
抽出したデータをCSV保存  

## 開発環境
- Windows 10 Home
- Python 3.9.4
- VS Code 1.55.0

## 仮想環境の作成
```
py-m venv .venv
```

## 仮想環境の有効化(Windows)
```
.venv\Scripts\activate
```

## 必要なパッケージのインストール
```
python -m pip install -r requirements.txt
```
